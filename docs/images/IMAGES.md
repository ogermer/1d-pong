# Image Assets

Place the following images in this directory:

## Required Images

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `wiring-diagram.png` | Wiring diagram showing ESP32, LED strip, and button connections | 800x600px |
| `project-photo-1.jpg` | Photo of the finished project (front view) | 1200x800px |
| `project-photo-2.jpg` | Photo of the finished project (detail/alternate view) | 1200x800px |

## Optional Images

| Filename | Description |
|----------|-------------|
| `gameplay.gif` | Animated GIF showing gameplay |
| `attract-mode.gif` | Animated GIF showing attract animations |
| `components.jpg` | Photo of all components before assembly |

## Image Guidelines

- Use PNG for diagrams and screenshots
- Use JPG for photos
- Use GIF for animations (keep file size reasonable, <5MB)
- Optimize images for web viewing
